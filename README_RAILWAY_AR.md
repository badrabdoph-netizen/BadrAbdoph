# نسخة جاهزة للنشر على Railway (Static + لوحة تحكم مؤقتة)

## الفكرة
- الموقع Static (Vite + React)
- لوحة التحكم `/admin` مؤقتًا بتخزن التعديلات في المتصفح (localStorage)
- مفيش قاعدة بيانات ولا Variables مطلوبة

## النشر على Railway
1) ارفع الملفات على GitHub
2) اربط الريبو في Railway واعمل Deploy
3) افتح الموقع
   - الصفحة الرئيسية: `/`
   - لوحة التحكم: `/admin`

## كلمة السر (مؤقتًا)
- كلمة السر الافتراضية للوحة التحكم: `admin`
- تقدر تغيرها لاحقًا من داخل الكود بسهولة (وهنعملها سوا)

## تشغيل محليًا
```bash
npm install
npm run dev
```
